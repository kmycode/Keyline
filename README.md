# Keyline
Keylineとは、フリーのストーリーエディタです。
ユーザは、登場人物などの要素を登録し、ストーリーを作っていくことができます。

![スクリーンショット](https://raw.githubusercontent.com/kmycode/Keyline/master/image/1.png)

現在、内部設計をまとめている段階です
